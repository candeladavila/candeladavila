## 💡 Featured Projects

### Kintsugi

A computer vision project focused on the automatic reconstruction of fragmented images.

The system divides an image into puzzle-like pieces, shuffles them and reconstructs the original image using different algorithms based on color analysis, edge detection, gradients and graph-based compatibility between pieces.

**Technologies:** Python, OpenCV, computer vision, image reconstruction, algorithms

---

### Full-Stack Development — BancoSol Campaign Management Platform

A full-stack web application developed for a Non-Governmental Organization, BancoSol, to organize food collection campaigns throughout the year.

The platform supports the management of campaigns, users, roles, stores, shifts and permissions. During the project, I worked on database design, backend and frontend development, client-server communication, API endpoints and role-based access control.

The application combines a Spring Boot backend with a React frontend and includes functionality adapted to different user profiles.

**Technologies:** Java, Spring Boot, React, JavaScript, HTML, CSS, SQL, JWT

---

### Research on Multiple Sclerosis Using Neural Networks

An ongoing research project focused on the analysis of multiple sclerosis using neural networks and computer vision techniques.

The project involves image processing, deep learning models and experimentation with neural network architectures to support the analysis of medical imaging data.

**Technologies:** Python, PyTorch, OpenCV, neural networks, computer vision
